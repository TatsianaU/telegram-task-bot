## Telegram-бот для командных задач (aiogram + SQLite)

Бот поддерживает команды:
- **/start** — приветствие и подсказка по командам.
- **/add `<текст>`** — добавить задачу в общую базу.
- **/list** — показать список задач.
- **/list_csv** — отправить список задач CSV-файлом.

### 1) Установка

Создайте и активируйте виртуальное окружение (пример для Windows PowerShell):

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

### 2) Настройка токена

В репозитории лежит файл `env.example`. Скопируйте его и переименуйте в `.env`, затем вставьте токен:

- **BOT_TOKEN**: токен от `@BotFather`
- **DB_PATH**: путь к файлу SQLite (по умолчанию `tasks.db`)

### 3) Запуск

```bash
python main.py
```

### 4) Пример использования

- `/add` → (бот попросит текст) → `Купить молоко`
- или одной командой: `/add Купить молоко`
- `/list`
- `/list_csv`
